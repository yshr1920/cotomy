# CotomyMetaElement

Wrapper for `<meta>` elements.

**Static Methods**

| Method | Description |
| --- | --- |
| `get(name)` | Returns the first matching meta element or an empty element. |

**Properties**

| Property | Type | Description |
| --- | --- | --- |
| `content` | `string` | Value of the `content` attribute. |
