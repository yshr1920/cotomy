# CotomyDebugSettings

Controls debug flags stored in `localStorage`.

**Methods**

| Method | Description |
| --- | --- |
| `isEnabled(key?)` | Returns `true` if a key (or global flag) is enabled. |
| `enable(key)` | Enables a specific debug key. |
| `disable(key)` | Disables a specific debug key. |
| `enableAll()` | Enables all debug output. |
| `disableAll()` | Disables all debug output. |
| `clear(key?)` | Clears a specific key or the global flag. |
